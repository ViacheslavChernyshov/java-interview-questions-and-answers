# Java Interview Questions and Answers українською

Збірка з **500+ Java interview questions and answers** для **Junior, Middle та Senior** Java backend розробників. Проєкт покриває **Spring Boot, Hibernate / JPA, SQL, PostgreSQL, Kafka, Docker, Kubernetes, Collections, Concurrency, Stream API, REST / HTTP, JVM memory, design patterns, transactions, microservices** та інші ключові теми, які регулярно з'являються на співбесідах.

Цей розділ робить українську версію повноцінною публічною точкою входу: не просто окремими перекладеними файлами, а зрозумілою картою контенту для швидкої підготовки та системного повторення.

## Чому цей репозиторій корисний

- Замість випадкового списку питань тут є **20 тематичних блоків**, з якими зручно готуватись по доменах
- Кожна тема підходить і для швидкого повторення, і для глибшого пропрацювання перед співбесідою
- Кожен файл можна використовувати як окрему навчальну картку
- Відповіді розкладені по трьох рівнях глибини: **Junior, Middle, Senior**
- Контент вже доступний **англійською, російською та українською**

## Для кого це

- Java backend розробників, які готуються до співбесід на рівні Junior, Middle або Senior
- Інженерів, яким потрібно швидко повторити Spring Boot, SQL, Kafka, concurrency або JVM internals
- Менторів та тимлідів, які хочуть дати команді структурований навчальний маршрут
- Кандидатів, яким зручніше вчитись по темах, а не по розрізнених нотатках

## Як краще користуватись

1. Виберіть один тематичний блок під конкретну ціль, наприклад Spring Boot, SQL, concurrency або Kafka.
2. Почніть з файлу `00. Навігатор по розділу.md`, щоб побачити структуру теми цілком.
3. Пройдіть питання по порядку, бо багато тем розкриваються шарами від бази до production-рівня.
4. Звіряйте свою відповідь з рівнями Junior, Middle та Senior, щоб калібрувати глибину.
